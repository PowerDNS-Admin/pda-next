# PDA Next

## TL;DR

To get started quickly with a simple deployment, execute the following commands on a *nix based system
with `git` installed:

```
git clone https://github.com/PowerDNS-Admin/pda-next.git
cd pda-next
source deployment/bare-metal/nix/setup.sh
```
